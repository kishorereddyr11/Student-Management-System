# Student Management System (Python)

A simple and efficient **Student Management System** built using **Python** that helps manage student records through an interactive console-based interface.

This project allows users to add, view, search, update, and delete student details easily.

---

## Overview

This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts in Python such as classes, objects, and methods.  
The system maintains student details including **name**, **roll number**, and **marks** in two subjects.

---

## Features

| Operation | Description |
|------------|-------------|
| Accept | Add a new student’s details — name, roll number, and marks for two subjects. |
| Display | Show the details of all students stored in the system. |
| Search | Search for a student using their roll number. |
| Delete | Remove a student record from the system by roll number. |
| Update | Update a student’s roll number with a new one. |
| Exit | Quit the program safely. |

---

## Operations Menu

When you run the program, you’ll get a menu like this:

1. Accept Student Details

2. Display Student Details

4. Delete Student Details
